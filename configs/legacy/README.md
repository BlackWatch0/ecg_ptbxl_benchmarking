# Archived Legacy Entrypoint Configurations

These YAML files preserve parameters from retired one-off scripts. They are
documentation artifacts, not taskmanager configurations and not executable
replacement runners. Use the supported workflows in `AGENTS.md` for current
training, evaluation, and reporting.

- `evaluate_exp0.yaml`: retired FastAI exp0 evaluation script.
- `cbam_emd_experiment.yaml`: retired SCP CBAM-EMD experiment script.
- `diagnose_cbam_emd*.yaml`: retired CBAM-EMD diagnostic scripts.
- `evaluate_cbam_emd_snr.yaml`: retired CBAM-EMD noisy-SNR evaluation script.
- `evaluate_noisy_mixed_*.yaml`: retired FastAI and Lightning noisy-mixed evaluation scripts.
